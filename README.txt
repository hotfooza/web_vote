=====================================================================
  ระบบเว็บแอปพลิเคชันโหวตคะแนน (Voting System)
=====================================================================

วิธีเปิดใช้งานระบบ:
---------------------------------------------------------------------
1. ดับเบิ้ลคลิกที่ไฟล์ "run.bat" เพื่อเริ่มต้นเปิดเซิร์ฟเวอร์
2. เปิดเบราว์เซอร์แล้วพิมพ์: http://localhost:3000

รหัสสำหรับทดสอบเข้าใช้งาน:
---------------------------------------------------------------------
- Admin (ผู้ดูแลระบบ): 00001
- User (สมาชิก 50 คนแรก): 10001 ถึง 10050
- รหัสยืนยัน Reset โหวต: 000001

ตำแหน่งไฟล์ฐานข้อมูล (สามารถก๊อปปี้ไปไว้ที่ไหนก็ได้):
---------------------------------------------------------------------
- web_vote/data/users.json       (ข้อมูลผู้ใช้งาน)
- web_vote/data/candidates.json  (ข้อมูลผู้ลงสมัคร)
- web_vote/data/votes.json       (ข้อมูลคะแนนโหวต)
